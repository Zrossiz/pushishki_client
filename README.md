# Клиент приложение для Pushishki.ru

## Технологии

- Next.JS
- Framer motion
- Sass

## Конфигурация .env

- API_URL адрес api
- FILESERVER_URL адрес файл сервера
- BOT_URL адрес бота
- CLIENT_URL адрес клиентского приложение
- API_KEY ключ api
- SERVER_IP адрес сервера

### Старт приложения

```bash
npm install 
npm run dev
```
